# Building Magia - バトルプロトタイプ

**Building Magia（ビルディング・マギア）**は、オフィスビルをモチーフにしたNFT対戦カードゲームです。

## 🎮 プレイ可能デモ

**👉 [今すぐプレイ](https://shimanto.github.io/building-magia-client/)**

## ✨ 特徴

- **3vs3のターン制バトル**：戦略的なカード選択で勝利を目指す
- **日本の密教をモチーフにしたキャラクター**：護法童子、緊那羅、迦楼羅などの式ビル
- **サイバーパンク × 和風**：未来的なネオンと伝統的な建築の融合
- **エナジーシステム**：限られたリソースで最適な行動を選択
- **NFT統合予定**：モナコイン（XMP）を使用したNFT発行機能を実装予定

## 🎯 ゲームシステム

### キャラクター（式ビル）

各キャラクターは9つのクラス（属性）を持ち、それぞれ強みと弱みがあります：

- **ゴホウ（護法）** - バランス型
- **キンナラ（緊那羅）** - 防御型
- **カルラ（迦楼羅）** - 攻撃型

### バトルの流れ

1. **カード選択**：手札から使用するカードを選択
2. **ターゲット指定**：攻撃対象の敵キャラクターを選択
3. **ダメージ計算**：カードの威力とキャラクターのステータスに基づいてダメージを算出
4. **敵のターン**：敵AIが自動で攻撃を実行
5. **勝敗判定**：どちらかのチームが全滅するまで繰り返し
